## 🎨 주요 기능

### 🏷️ 브랜드 스토리 페이지 (`/brand/`)

<div align="center">

| 기능               | 설명                                      |
| ------------------ | ----------------------------------------- |
| 📖 **브랜드 철학** | 스웨디시 뉴트라의 브랜드 철학과 역사 소개 |
| ✨ **애니메이션**  | 부드러운 스크롤 애니메이션 효과           |
| 📱 **반응형**      | PC/태블릿/모바일 완벽 대응                |

</div>

**🔍 주요 섹션:**

- 🎯 **히어로 섹션** - 브랜드 첫인상
- 🏆 **품질의 약속** - 스웨덴 말뫼의 품질 기준
- 🌍 **NORDIC 라인** - 북유럽에서 세계로
- 💭 **운영 철학** - 가족경영의 세심함
- 🌱 **지속 가능한 선택** - 환경을 생각하는 선택
- ❤️ **건강에 대한 생각** - 스웨디시 뉴트라의 건강관
- 🏢 **동도F&P 소개** - 한국 독점 공급사
